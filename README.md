# Quantum Neural Network vs Classical Neural Network

This project compares the performance of a Classical Neural Network (CNN) and a Quantum Neural Network (QNN) for phishing website detection using machine learning and quantum computing techniques.

## Project Overview
The notebook explores the integration of Quantum Machine Learning (QML) with traditional deep learning models for binary classification tasks.

The project includes:
- Dataset preprocessing and feature engineering
- Feature selection using SelectKBest
- Classical Neural Network implementation with TensorFlow/Keras
- Quantum Neural Network implementation using PennyLane
- Model training and evaluation
- Accuracy and loss visualization
- Performance comparison between CNN and QNN models

The phishing dataset was used to classify websites as legitimate or phishing based on selected features.
